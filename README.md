# 1.5L DOHC 高转速发动机正时驱动系统设计

## 项目概述

针对一台**直列4缸、1.5L排量、双顶置凸轮轴（DOHC）发动机**的正时驱动系统进行设计。发动机最高转速 **18000rpm**，功率85kW。

## 设计成果

经过三轮迭代，最终方案为：

```
主传动: 静音链 CR(19T奇数) → EX(38T)  6.35mm节距 110节 链速36.4m/s
副传动: 短链   EX(30T)    → IN(30T)  6.35mm节距 60节  链速28.6m/s
```

## 设计依据

方案直接对标以下量产高转速发动机的正时驱动系统：

| 参考机型 | 最高转速 | 链型 | 节距 | 曲轴齿 | 链速 |
|---------|:-------:|:----:|:----:|:-----:|:---:|
| Kawasaki ZX-25R | 17000rpm | 静音链 | 6.35mm | 20T | 36.1m/s |
| Honda CBR600RR | 15200rpm | 静音链 | 6.35mm | 21T | 33.9m/s |
| Honda CBR1000RR-R | 14500rpm | 静音链 | 6.35mm | 22T | 33.9m/s |
| Yamaha YZF-R1 | 13500rpm | 静音链 | 6.35mm | 22T | 31.5m/s |
| Suzuki GSX-R600 | 14500rpm | 静音链 | 6.35mm | 21T | 33.8m/s |
| BMW S1000RR | 14500rpm | 静音链 | 6.35mm | 22T | 33.9m/s |
