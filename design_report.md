## ■ 重要更正

**载荷修正:**
本系统驱动的是**凸轮轴（气门弹簧）**，非发动机输出轴。

| 项目 | 错误值 | 正确值 |
|:----|:------:|:------:|
| 驱动对象 | 发动机输出 | **凸轮轴** |
| 大齿轴扭矩 | ~155 Nm | **~29 Nm** |
| 圆周力 Ft | ~7,542 N | **~707 N** |
| 结论 | 必须锻钢 | **标准PM即可** ✅ |

## ■ 齿轮与轴承配合方案

### 配合尺寸

| 配合面 | 公差代号 | 数值 |
|:------|:-------:|:----:|
| 轴承内圈 ↔ 轴 | φ25j6 | +0.002~+0.013 mm |
| 轴承外圈 ↔ 齿轮孔 | φ52H7 | 0~+0.025 mm |
| 轴颈粗糙度 | Ra ≤ 0.4 μm | |
| 轴承孔粗糙度 | Ra ≤ 0.8 μm | |

### 装配方式（两轴相同）

固定轴（螺栓固定在壳体上）→ 轴承内圈 φ25j6 过盈在轴上 → 轴承外圈 φ52H7 压入齿轮内孔 → 齿轮+轴承在固定轴上旋转。锁紧螺母轴向固定。
