# 1.5L DOHC 高转速发动机正时驱动系统 — 设计报告

## 1 概述

直列4缸、1.5L、DOHC、18000rpm、85kW。曲轴坐标(0,0)，排气凸轮(-49,253.53)，进气凸轮(47,253.53)。

---

## 2 对标验证

| 参考机型 | 转速 | 链型 | 节距 | 曲轴齿 | 布局 | 链速 |
|:---|:---:|:---:|:---:|:---:|:---|:---:|
| Kawasaki ZX-25R | 17000 | 静音链 | 6.35mm | 20T | 单链+短链/齿轮 | 36.1 m/s |
| Honda CBR600RR | 15200 | 静音链 | 6.35mm | 21T | 单链+齿轮 | 33.9 m/s |
| Yamaha YZF-R1 | 13500 | 静音链 | 6.35mm | 22T | 单链+齿轮 | 31.5 m/s |
| **本设计** | **18000** | **静音链** | **6.35mm** | **19T** | **单链+短链** | **36.4 m/s** |

核心技术路线与ZX-25R一致，链速与ZX-25R持平。

## 3 布局方案

```
【静音链主传动】                 【短链副传动】
  CR(19T)                        EX(30T)
    │                              │
    │  ← 紧边(驱动)                │  ← 90°啮合
    │                              │
    ▼                              ▼
  EX(38T) ──────────────→ IN(30T)
         短链 60节×381mm
         同向旋转 ✅
```

**为什么19T不是20T或21T？**
- 19T奇数 → 磨损自均匀 ✅
- 21T → 链速40.2m/s，超过Morse TEC 40m/s上限 ⚠️
- 19T → 链速36.4m/s，与ZX-25R(36.1m/s)同等安全裕度
- 19T比21T张力大10%(2338N vs 2117N)，12mm宽静音链完全承受

## 4 详细参数

### 4.1 主传动 — 静音链 CR(19T) → EX(38T)

| 参数 | 数值 |
|------|------|
| 链条类型 | BorgWarner Morse TEC 静音链（倒齿链） |
| 节距 | 6.35 mm (1/4") |
| 曲轴链轮 | 19T（奇数 ✅），节圆 φ38.58 mm，齿顶圆 φ41.1 mm |
| 排气凸轮链轮 | 38T，节圆 φ76.90 mm，齿顶圆 φ79.4 mm |
| 传动比 | 2:1 ✅ |
| 链节数 | **110 节** |
| 链条长度 | **698.5 mm** |
| 链速 @ 18000rpm | **36.36 m/s** ✅ |
| 曲轴包角 | **171.5°** ✅ |
| 紧边张力 | 2338 N (238 kgf) |
| 设计安全系数 | 4.0 |
| 破断载荷要求 | ≥9351 N |

### 4.2 副传动 — 短链 EX(30T) → IN(30T)

| 参数 | 数值 |
|------|------|
| 链条类型 | 同主链，Morse TEC 静音链 6.35mm |
| 链轮 | 30T × 2（等径），节圆 φ60.75 mm |
| 传动比 | 1:1（两凸轮同速同向 ✅） |
| 中心距 | 96 mm |
| 链节数 | **60 节** |
| 链条长度 | **381.0 mm** |
| 链速 @ 9000rpm | 28.63 m/s |
| 包角 | 180° × 2（等径链轮） |
| 紧边张力 | 远低于主链（仅克服气门弹簧力矩） |

### 4.3 排气凸轮轴集成

```
轴向布局（从发动机前端面看）：
┌──────┐  ┌────┐  ┌──────┐
│38T   │  │隔圈│  │30T   │
│主链轮│──│5mm │──│副链轮│
│φ76.9 │  │    │  │φ60.7 │
└──────┘  └────┘  └──────┘
   ~12mm     5mm     ~10mm
   └────────── 总宽 ~27mm ──────────┘
```

两个链轮可一体加工（阶梯式链轮坯）或分体键连接。

## 7 润滑系统

| 喷嘴 | 位置 | 目标 |
|------|------|------|
| N₁ | 曲轴链轮紧边啮入点 | 主链与曲轴链轮啮合区 |
| N₂ | 排气凸轮轴链轮齿面 | 主链与38T链轮啮合区 |
| N₃ | 短链啮合区 | 30T→30T短链传动 |

- 喷嘴孔径：φ1.0 mm
- 流量：0.4 L/min/喷嘴 @ 3 bar
- 机油：全合成 5W-50 或 10W-60，HTHS≥3.5 cP

## 8 风险与对策

| 风险 | 对策 |
|:---|:---|
| 链板高周疲劳（6600Hz交变） | 喷丸强化+超精销孔，安全系数4.0 |
| 链速接近40m/s上限 | 选用19T降至36.4m/s；DLC销轴涂层 |
| 短链伸长 | 60节短链，总伸长量小，6mm行程足够 |
| 副传动链条磨损 | 与主链同规格，便于备件统一 |
| 张紧器响应不足 | 大流量单向阀+高压蓄能器 |
| 热负荷 | 3个定向喷嘴冷却+油底壳散热鳍片 |
