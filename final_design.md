# 直列四缸 DOHC 正时驱动系统 — 最终设计方案

## 1. 系统概述

| 项目 | 参数 |
|:----|:----:|
| 发动机 | 直列四缸，双顶置凸轮轴 (DOHC) |
| 最高转速 | 18,000 RPM |
| 减速方式 | 齿轮两级减速 (2:1) + 链条等速传动 (1:1) |
| 齿轮模数 | mₙ = 1.75 (法向) |
| 螺旋角 | β = 12° |
| 布局 | 曲轴→惰轮→大齿轮中心在一条直线上 |

**齿轮减速路径：**
```
曲轴 23T ——→ 惰轮 33T ——→ 大齿+链轮 46T (同轴)
  18,000 RPM      12,545 RPM        9,000 RPM
```

**链条传动路径：**
```
大齿轴链轮 29T (9,000 RPM)
     ├──→ 进气凸轮链轮 29T (9,000 RPM)
     └──→ 排气凸轮链轮 29T (9,000 RPM)
```

### 6.3 转速校核

| 轴 | 实际转速 | 极限转速 | 裕量 | 结论 |
|:--|:-------:|:--------:|:---:|:----:|
| 惰轮轴 | **12,545 RPM** | 17,000+ RPM | **≥ 26%** | ✅ **充足** |
| 大齿+链轮轴 | **9,000 RPM** | 17,000+ RPM | **≥ 47%** | ✅ **充足** |

### 6.4 单轴承可行性分析

**惰轮轴 — 单轴承 ✅ 可行**

- 齿轮啮合力：曲轴和惰轮的啮合力在**同一直线上方向相反**
- 径向合力 ≈ 两力之差（非之和）→ 轴承径向载荷可控
- 轴向力 Fₐ = Fₜ × tan(12°) ≈ 0.213Fₜ → 双列角接触天生适合双向承载
- 齿轮紧靠轴承安装 → 悬臂短 → 弯矩小
- 配合方式：轴承外圈固定于壳体，齿轮安装于轴伸端

**大齿+链轮轴 — 单轴承 ✅ 可行**

- 齿轮和链轮**分居轴承两侧** → 弯矩相互抵消
- 齿轮啮合力向上，链条拉力也向上 → 合力方向一致，无交变弯矩
- 转速仅 9,000 RPM → 轴承寿命长
- 安装方式：轴承居中，齿轮在一侧，链轮在另一侧

## 9. 关键设计总结

```
齿轮减速: 2:1 (23T → 46T，惰轮33T中间过渡)
齿轮模数: m = 1.75, β = 12°
变位系数: 全部 x = 0 (标准齿轮)

坐标:
  曲轴      23T   (0, 0)
  惰轮      33T   (9.50, 49.18)
  大齿+链轮 46T   (22.91, 118.57)

链轮: 全部 29T, p = 6.35mm, ~90节
链条线速度: 27.7 m/s ✅

轴承: 3205A × 2个 (惰轮轴1个 + 大齿轴1个)
     固定轴方案: 两根轴均螺栓固定在壳体上, 齿轮+链轮一体件在轴承外圈旋转
     取消键连接! 大齿轮与链轮一体加工, 无限疲劳寿命, 完美动平衡
惰轮轴转速: 12,545 RPM (裕量26%)
大齿轴转速: 9,000 RPM (裕量47%)

**载荷更正: 正时系统驱动凸轮轴(气门弹簧), 非发动机输出, 扭矩仅29Nm**
齿根应力: 曲轴124MPa / 惰轮58MPa(交变) / 大齿83MPa — 全部远低于PM疲劳极限200MPa ✅

材料: 全齿轮标准粉末冶金PM FLN2-4408渗碳淬火
      曲轴14mm / 惰轮21mm / 大齿21mm (轴承决定齿宽,非应力)
链轮: 40Cr高频淬火48-55HRC
```
